命令名称： Cjson.Decode json数据转换table
命令功能： json数据转换table
命令参数：
参数1【必要】：字符串, json字符串
返回值： 键值表：返回json字符串所对应的键值表
脚本例子：
按键精灵手机版语法 
Import "Cjson.lua"
Dim json, table
json = "{""age"":""23"",""testArray"":{""array"":[8,9,11,14,25]},""Himi"":""himigame.com""}"
// json数据转换table
table = Cjson.Decode(json)
// 输出table内容
For Each k, v In table
    TracePrint k, v
Next
// table数据转换json
json = Cjson.Encode(table)
TracePrint json
命令名称： Cjson.Encode table数据转换json
命令功能： table数据转换json
